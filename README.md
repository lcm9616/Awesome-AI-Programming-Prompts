## 📖 项目简介

本项目是一个专门针对**编程领域**的 AI 提示词集合，旨在为开发者提供高质量、实用的提示词模板。无论你是初学者还是资深开发者，都能在这里找到适合的提示词来提升你的开发效率。

### 🌟 项目特色

- 🎯 **专业性**: 专注编程领域，提供精准的技术指导
- 🔄 **持续更新**: 定期添加新的提示词和改进现有内容
- 🌍 **多语言支持**: 提供中文和英文双语版本
- 🤖 **多模型兼容**: 支持主流 AI 模型，确保最佳体验
- 📚 **结构化**: 采用标准化格式，易于理解和使用

> 💡 **提示**: 如果你对 AI 智能体开发不感兴趣，可以直接跳过本项目。
> 
> 🚧 **开发进度**: 本项目正在持续完善中，更多编程语言和框架的提示词即将推出！

## 🚀 快速开始

### 📋 使用方法

1. **选择合适的提示词**: 根据你的开发需求，从上述分类中选择对应的提示词文件
2. **复制提示词内容**: 打开对应的 `.txt` 文件，复制其中的提示词内容
3. **粘贴到 AI 工具**: 将提示词粘贴到你使用的 AI 编程助手中
4. **开始高效编程**: 享受 AI 助手为你提供的专业编程指导

### 💡 使用技巧

- 🎯 **明确需求**: 在使用提示词前，先明确你的具体开发需求
- 🔄 **适当调整**: 根据项目特点，适当调整提示词内容
- 📝 **保存常用**: 将常用的提示词保存到你的代码片段库中
- 🤝 **分享经验**: 欢迎分享你的使用心得和改进建议
- 🔔 **关注更新**: 定期查看项目更新，获取最新的提示词内容
- 📖 **阅读文档**: 查看 [使用指南](./docs/usage-guide.md) 获取详细的使用说明
